## 📥 Fichiers GPX

Tous les GPX sont dans le dossier [`/gpx`](./gpx/).  
Les tracés sont générés algorithmiquement depuis les coordonnées réelles.  
Pour les tracés précis, utiliser les **liens Komoot** dans le site web.

---

## 🛠️ Génération

```bash
# Générer les GPX
python3 generate_gpx.py

# Générer la page web
python3 generate_page.py
```
